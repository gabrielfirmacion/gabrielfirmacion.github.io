Almost every class needs to start with the method initialize(line 2) which takes an argument from the player. I made 'choice', my instance variable which is why it has '@' before it (line 3). You make an instance variable every time you want your variable to be called from more than 1 method inside your class. If I did not make 'choice' an instance variable, I will not be able to call it from my 'check' method.

As you can see from my 'check' method (lines 6 to 21). I called the instance variable 'choice' a lot of times. That instance variable will determine whether or not you won the game.
